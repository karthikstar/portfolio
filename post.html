Smartbrain App Project

https://smartbrain-karthik.herokuapp.com

This was my first full-stack web development project I did back in 2020, with the guidance of online tutorials.

The key components of this app are the registration form, login authentication, and the face detection feature in the app provided by a 3rd party API.

Once the user logs in to the app, they can upload a image with a face from online sources, and paste the url in the field given. The app will detect if there are any faces in the picture and put a square around it, and update the score.

Technologies:
  • - HTML & CSS
  • - Javascript
  • - React JS
  • - Postgres
  • - Heroku
  • - Clarifai API
