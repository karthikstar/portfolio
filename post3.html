Real-Time Chat App

https://realtimechat-react.netlify.app

This is a realtime chat application made using React, Express, Node and Socket.io. I made this because i wanted to learn more about the technologies which enable live interactions between different people. It will also be useful for me in the future when i go on to develop other projects such as E-Commerce platforms, where such real-time chat applications could be deployed on.

The key components of this app are the login field and the chat interace.

A user can enter a name and a random room number to join a room. To invite another person, he or she simply needs to tell the other person to key in the same room number and they can start chatting in real-time.

Technologies:
  • - HTML & CSS
  • - Javascript
  • - React JS
  • - Express
  • - Socket.io
  • - Netlify
